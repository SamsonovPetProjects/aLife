#### Цель проекта

Создать симуляцию для реализации и обучения DL алгоритмов, с целью повышения технических навыков и углубления в материал.

#### Архитектура: 

1) При помощи библиотеки PyGame создана симуляция - поле n * m клеток 
2) По полю в случайном порядке расставлена "еда" - один из основных мотиваторов (Reward) сети в обучении с подкреплением (Reinforcement Learning)
3) В симуляции агент испытывает "голод", и умирает без "еды" (в разработке находится генетический алгоритм (Genetic Algorithm)) 
4) Всё поле PyGame векторизовано через Numpy, что значительно повышает общую производительность системы: срезы зон видимости, тепловые карты, перемещения и т.д.
5) CNN - агент воспринимает окружающий мир через два слоя свёрток. В проекте CNN слои выступают в роли Encoder
6) RNN - Hidden state позволяют реализовать память (использована GRU архитектура)
7) __!В разработке__ создание тини трансформера (Tiny Transformer) и настройка механизма самовнимания (Self-Attention)

#### Текущие задачи и R&D

Интеграция Tiny Transformer: Замена или дополнение GRU-слоя механизмом Self-Attention для анализа глобальных зависимостей на поле.

#### Описание функционала по файлам

__main.py__
- Этот файл является входной точкой симуляции. Он объединяет работу всех модулей проекта

__config.py__ 
- Содержит все гиперпараметры проекта, включая размеры поля, настройки наград и параметры нейросетей, обеспечивая единую точку управления конфигурацией симуляции.

__simulation.py__ 
- Управляет жизненным циклом симуляции, отвечая за выбор архитектуры нейросети (CNN или RNN), инициализацию графического окна и координацию процессов обучения агентов в реальном времени. Содержит основной цикл, в котором происходит сбор данных в буферы памяти и выполнение шагов градиентного спуска для обновления весов моделей.

__environment.py__ 
- Описывает физику мира в виде двумерной матрицы Numpy, предоставляя методы для размещения ресурсов и обработки взаимодействий агентов с объектами. Рассчитывает систему вознаграждений (Reward) и управляет жизненными показателями сущностей, включая затраты энергии на перемещение и смерть от голода.

__entities/agent.py__ 
- Определяет класс агента, его физические характеристики, систему "голода" и интерфейс взаимодействия с нейронной сетью.

__brain/model.py__
- Базовая архитектура нейронной сети, выполняющая роль "мозга" агента для принятия решений на основе текущего состояния среды.

__brain/recurrent_model.py__ 
- Рекуррентная модификация нейросети на базе GRU, которая позволяет агенту хранить скрытое состояние и учитывать историю предыдущих событий.

__brain/replay_buffer.py__ 
- Механизм циклической памяти, который хранит прошлый опыт агента для стабилизации процесса обучения через случайные выборки данных.

__brain/sequential_buffer.py__ 
- Специализированный буфер для работы с временными рядами, сохраняющий правильную последовательность кадров для обучения рекуррентных слоев.
